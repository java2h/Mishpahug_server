# Mishpahug_server
Appointment scheduling application

Installation and startup:
  Ensure MySQL is running on port 3306;
  Select any folder and open the terminal;
In the terminal:
  git clone
  cd MishpahugHaifa
  mvn spring-boot:run
  cd ../MishpahugHaifaFront
  npm install
  npm start
Open browser and go to localhost:8080
